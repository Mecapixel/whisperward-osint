# WhisperWard OSINT - Development Roadmap

## Project Vision
A robust, ethical, and technically excellent defensive OSINT toolkit
for online safety investigations.

- [ ] Core CLI with Typer + Rich (whisperward.py)
- [ ] SQLite database schema (schema.sql) and DatabaseManager
- [ ] BaseOSINTModule abstract class
- [ ] Evidence Packager with chain-of-custody logging
- [ ] Project documentation (README, DISCLAIMER, ACCEPTABLE_USE)

**Target:** May/June 2026
